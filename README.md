# WordPress Selenium Automation

This project automates basic functionality of the WordPress website using Python, Selenium, and Pytest.

## Features
- Launch WordPress website
- Verify page title
- Hover on "Download & Extend"
- Click on "Themes"
- Search for a theme (Astra)
- Validate theme results

## Tech Stack
- Python
- Selenium WebDriver
- Pytest

## Run Tests
```bash
python -m pytest -v
